More Tools & Wallets

Hier findest du weitere nützliche Werkzeuge. Wähle eines aus und richte es am besten im Brave Browser ein. Alle Links führen zur Original-Seite bzw. in die App-Stores und öffnen in einem neuen Tab.

Here you find additional useful tools. Pick one and set it up in the Brave browser. All links go to the official sites or app stores and open in a new tab.

Rabby Wallet Empfehlung für Desktop Recommended for desktop

Sichere, EVM-fokussierte Browser-Wallet (DeBank). Gute Netzwerk-Erkennung, Anti-Phishing-Checks, klare Transaktionsvorschau.

Secure, EVM-focused browser wallet (by DeBank). Great network detection, anti-phishing checks, and clear transaction previews.

rabby.io Chrome/Brave Edge
  • Installation: Über Web-Store hinzufügen → Wallet erstellen/Seed importieren → Netzwerke prüfen.
  • Install: Add via web store → create/import wallet → verify networks.
MetaMask

Der Klassiker. Browser-Extension + Mobile App, breite Dapp-Unterstützung. Ideal, wenn viele Anleitungen darauf verweisen.

The classic choice. Browser extension + mobile app, wide Dapp support. Handy because many guides reference it.

metamask.io Chrome/Brave Firefox iOS Android
  • Installation: Erweiterung/App → neues Wallet oder Seed importieren → Sicherung testen → Netzwerke hinzufügen.
  • Install: Extension/app → create or import seed → verify backup → add needed networks.
Rainbow

Schöne Mobile-Wallet mit Fokus auf Einfachheit, Preis-Infos und NFTs. Ideal fürs erste On-Chain-Erlebnis am Smartphone.

Polished mobile wallet focused on simplicity, prices, and NFTs. Great for first on-chain steps on a phone.

rainbow.me iOS Android
  • App installieren → Wallet erzeugen → Seed offline sichern → Watchlist/Netzwerke einstellen.
  • Install app → create wallet → back up seed offline → set watchlist/networks.
Coinbase Wallet

Eigenständige Wallet getrennt vom Exchange-Konto. Gute Dapp-Integration, Extension + Mobile.

Standalone wallet (separate from exchange account). Solid Dapp integration, extension + mobile.

wallet.coinbase.com Chrome/Brave iOS Android
  • Extension/App installieren → Wallet verbinden → Signaturen testen → Benachrichtigungen konfigurieren.
  • Install extension/app → connect wallet → test signing → set notifications.
WalletConnect (Protokoll)

Kein Wallet, sondern das Verbindungs-Protokoll zwischen Dapps und Wallets (QR-Code/Deep-Link). Schon in vielen Apps integriert.

Not a wallet but the connection protocol between Dapps and wallets (QR/deep-link). Built into many apps.

walletconnect.com
  • Nutzen: In der Dapp „WalletConnect“ wählen → QR mit deiner Wallet scannen → Prüfung der Berechtigungen.
  • How-to: Choose “WalletConnect” in the dapp → scan QR with your wallet → review permissions.
Tangem (Hardware-Wallet, NFC)

Seed-freie Karten-Wallet mit sicherem Element (NFC). Ideal für Einsteiger, die Hardware-Sicherheit ohne Seed-Zettel bevorzugen.

Seed-less card wallet (NFC secure element). Great for beginners wanting hardware-grade security without seed paper.

tangem.com iOS App Android App
  • App installieren → Karte per NFC koppeln → Backup-Karten einrichten → Test-Transaktion senden.
  • Install app → pair card via NFC → set backup cards → send a test transaction.
M-PESA

Mobile-Money-Standard in vielen afrikanischen Ländern (Safaricom/Vodafone). Nützlich für Fiat-↔-Wallet Brücken vor Ort.

Mobile money standard in many African countries (Safaricom/Vodafone). Useful for local fiat ↔ wallet bridges.

iOS Android Info
  • App aus Store installieren → Registrierung/KYC laut Vorgaben → Kontakte/USSD-Funktionen prüfen.
  • Install from store → register/KYC per provider → check contacts/USSD features.
